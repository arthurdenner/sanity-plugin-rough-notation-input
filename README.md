# sanity-plugin-rough-notation-input

A wrapper for react-rough-notation-input

## Installation

```
sanity install rough-notation-input
```

## Configuration

The plugin can be configured through `<your-studio-folder>/config/rough-notation-input.json`:

```json
{
  "add-config": "here"
}
```
